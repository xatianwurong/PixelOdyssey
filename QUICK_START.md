## ✨ 设计系统核心内容

### 字体

```
Body (13pt):          正文文本
Headline (18pt):      组件标题
Subtitle (15pt):      小标题
Display (28pt):       页面标题

所有文字都 ≥ 11pt (无障碍标准)
```

### 颜色

```
Primary (#3278FF):      主色 (蓝色)
Surface (#252525):      表面色
Text Primary:           主文字
Success/Warning/Error:  功能色

所有对比度 ≥ 4.5:1 (AA 级)
```

### 间距

```
S:   8px   (项间距)
M:   16px  (标准间距,最常用)
L:   24px  (大间距)

所有间距都是 8px 的倍数
```

### 控件

```
Button:      44px (标准)
List Item:   44px (标准)
圆角:        8px
边框:        1px

所有触点 ≥ 44px (WCAG 标准)
```
