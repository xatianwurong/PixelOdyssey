## 🔴 High Priority - 必须做（第 1-3 周）

### Phase 1: 基础设施（Week 1-2）

#### [ ] Task 1.1: 服务注入容器
- **描述**: 创建 ServiceContainer 轻量级 DI 框架
- **文件**: `services/ServiceContainer.h`, `services/ServiceProvider.h`
- **预期工作量**: 4-6 小时
- **依赖**: 无
- **验收标准**:
  - [ ] 编译通过，无警告
  - [ ] 可以注册和获取服务
  - [ ] 支持单例和工厂模式
  - [ ] 单元测试覆盖 80%+

**实施步骤**:
1. 创建 ServiceContainer.h（复制模板）
2. 创建具体服务实现（ILogger, IColorScheme, IToolService）
3. 在 MainWindow 中初始化服务
4. 替换所有 ColorScheme::Instance() 为依赖注入
5. 编写单元测试

**风险**: 大量替换可能引入回归，需要充分测试
