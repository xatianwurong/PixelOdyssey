## 🎨 设计系统核心内容

### 1️⃣ 字体系统 (Typography)

**8 个层级** (改进前: 3 个)

```
Display Large  → 32pt, Bold      (页面主标题)
Display        → 28pt, Bold      (重要标题)
Headline       → 18pt, SemiBold  (组件标题)
Subtitle       → 15pt, Medium    (小标题)
Body Large     → 14pt, Medium    (强调文本)
Body           → 13pt, Normal    (正文 ⭐)
Body Small     → 12pt, Normal    (辅助文字)
Caption        → 11pt, Normal    (说明)
```

**字体粗细**: Normal, Medium, SemiBold, Bold (4 个等级)  
**行高规范**: 1.2, 1.5, 1.75, 2.0  
**无障碍**: 所有文字 ≥ 11pt ✅

### 2️⃣ 颜色系统 (Color System)

**3 个主题** (改进前: 1 个)

```
深色主题 (Dark)        → 45+ 色值
浅色主题 (Light)       → 45+ 色值
高对比度 (HighContrast) → 简化配置
```

**色值组织**:
- 主色系: Primary 50-700 (8 个梯度)
- 中性色: Neutral 50-900 (10 个梯度)
- 功能色: Success, Warning, Error, Info (4 个)

**对比度**: 全部 ≥ 4.5:1 (WCAG AA) ✅

### 3️⃣ 间距系统 (Spacing)

**8px 网格系统** (改进前: 无规范)

```
XS   → 4px
S    → 8px
M    → 16px  (⭐ 标准,最常用)
L    → 24px
XL   → 32px
XXL  → 48px
```

**应用**: 所有间距都是 8px 倍数 ✅

### 4️⃣ 控件规范 (Components)

**高度规范**:

| 控件 | 改进前 | 改进后 | WCAG |
|------|-------|-------|------|
| 按钮 | 38px | 44px | ✅ |
| 列表项 | 32px | 44px | ✅ |
| 工具栏 | 56px | 56px | ✅ |
| 图标 | 16px | 24px | ✅ |

**触点大小**: 所有 ≥ 44px × 44px ✅

### 5️⃣ 状态系统 (States)

```
DEFAULT   → 默认状态
HOVER     → 背景 +8% 白色
ACTIVE    → 背景 +12% 黑色
FOCUSED   → 2px 焦点环
DISABLED  → 文字变淡 (38%)
ERROR     → 红色边框
SUCCESS   → 绿色边框
WARNING   → 橙色边框
```

### 6️⃣ 动画系统 (Animation)

```
Fast     → 100ms (UI 反馈)
Standard → 200ms (标准过渡)
Slow     → 300ms (进入/退出)

Easing: Linear, EaseInOut, EaseIn, EaseOut
```

### 7️⃣ 无障碍设计 (Accessibility)

**WCAG 2.1 AA+ 全面支持** ✅

- 颜色对比度 ≥ 4.5:1
- 触点大小 ≥ 44px
- 焦点指示清晰
- 键盘完全支持
- 屏幕阅读器友好
- 文本可缩放
- 动画可禁用
